# WashPro SFV — Landing Page

Premium landing page for WashPro SFV, a pressure washing company based in the San Fernando Valley, California.

## Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript (no frameworks, no jQuery)
- Google Fonts: Bebas Neue, Syne, DM Sans
- Placeholder images via picsum.photos

## Features

- Fixed nav with transparent → blur-dark scroll transition
- Full-viewport hero with animated drip lines, pulsing orb, and stroke headline
- Scroll-triggered count-up stats bar
- 3-column masonry portfolio grid with lightbox modal
- 6 service cards with hover animations
- Interactive before/after drag-to-reveal comparison slider (mouse + touch)
- Why WashPro section with animated numbered reasons
- 3-column testimonials with star ratings
- CTA section with green radial glow
- Floating phone button (fades in at 1.5s, floats persistently)
- Mobile hamburger menu with full-screen overlay
- IntersectionObserver scroll reveals on all sections and cards
- Fully responsive: desktop / tablet / mobile breakpoints
